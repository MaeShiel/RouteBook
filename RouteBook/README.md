# RouteBook

RouteBook is a modern Android app for managing delivery routes. It allows users to:
- View a list of cities in their delivery route
- See stops for each city
- Navigate to stops using Google Maps
- Import/export route data as .csv files
- Add cities and stops manually
- Enjoy a modern Material Design UI

## Features
- Home page with city list (shows city name, last update, and number of stops)
- Expand city to view stops
- Tap stop to open Google Maps navigation
- Import/Export .csv functionality
- Add cities and stops manually
- Hamburger menu in top bar

## Getting Started
1. Open the project in Android Studio or VS Code with Android development extensions.
2. Build and run the app on an emulator or device.

## File Import/Export
- Import: Tap 'Import' and select a .csv file with your route data.
- Export: Tap 'Export' to save your current route data as a .csv file.
